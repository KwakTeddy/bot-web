想要制作什么样的聊天机器人?

PlayChat对话的基本单位是一个用户的问题和一个聊天机器人的回答，通过这样的方式来进行连续性的对话。在PlayChat上制作聊天机器人主要可以通过2种方法。

对话集聊天机器人

这是一种让聊天机器人学习对话集的制作方式，学习的对话集将问题和答案作为一个数据单元(对话模块)。FAQ聊天机器人主要使用这种制作方法。

对话场景聊天机器人

这是一种使用PlayChat提供的对话场景图来制作聊天机器人的方法。在使用对话场景时可以不仅仅局限于一对一的问答方式，而同时可以实现具有特定流程和特定功能的对话(爬虫，验证手机号码，连接外部服务器等)。

Tip

可以同时使用对话集和对话场景来制作聊天机器人，通过这样的方式能制作出表现更自然，实用性更强的聊天机器人。您最好是根据想要通过聊天机器人来达到什么样的目的来决定是基于对话集制作聊天机器人，还是基于对话场景来制作聊天机器人。

制作对话集聊天机器人

在这部分内容将使用 对话集来制作一个简单的 公司介绍聊天机器人，公司介绍聊天机器人能回答公司名称，电话号码，主要做什么，公司位置等问题。

使用对话集来制作聊天机器人需要让聊天机器人学习以问题和答案组成的数据单元(对话模块)，多个问题和答案组成的对话模块可集合成对话集，同时您也可以让聊天机器人学习多个对话集。

制作聊天机器人

按照以下步骤创建聊天机器人。

  1. 如果您还没有PlayChat账号，请先进行注册。如果您已经有了PlayChat账号，请进行登录。
  2. 在机器人管理画面创建聊天机器人 -> 选择空白机器人。

  3. 输入需要填写的项。
  4. 点击"保存"按钮。创建完一个新的聊天机器人后，新的聊天机器人将会以卡片的形式显示在聊天机器人管理画面，然后单击选择创建好的聊天机器人。

学习对话模块

对话模块默认是把一个问题和一个答案连接到一起，是一对一的连接方式。但对话模块同时也支持把多个问题和多个问题连接在一起的多对多连接方式。

按照以下步骤让聊天机器人学习对话模块。

  1. 在页面左侧菜单中选择"开发" -> "对话集"。
  2. 在问题栏中输入聊天机器人用户可能会问到的问题，在答案栏中输入您想让聊天机器人给出的回答。
    • 用户问题 - "你们公司在哪里？"
    • 机器人回答 - "我们公司Playchat在江南地铁站6号出口的宏宇大楼。"
  3. 点击"添加"按钮。

  4. 测试
    现在聊天机器人已经理解了您输入的问题，并随时准备着回答用户的相关问题。您现在可以试着在页面右侧的测试对话框中输入您之前添加的问题，您也可以输入和这个问题比较类似的问题，因为PlayChat可以回答与添加过的问题类似的问题。

Tip

页面底部是一个实时对话分析窗口，主要是对以用户输入和聊天机器人回答为基本单元的对话进行分析。

  • 用户输入分析 - 对聊天机器人用户的输入值进行自然语言处理并显示结果。
  • 聊天机器人回答分析 - 显示了聊天机器人用户的输入值和哪些数据能匹配，以及聊天机器人是否给出了回答。

学习多对多对话模块

按照以下步骤让聊天机器人学习由多个问题和多个回答连接成的对话模块。

  1. 在问题栏中输入聊天机器人用户可能会问到的问题，通过点击问题栏下方的"+"按钮来增加多个问题。

    • 用户问题 - "你们公司的电话号码是多少？", "我想知道你们的电话号码", "跟我说一下你们公司的电话号码"

  2. 在答案栏中输入聊天机器人应该给出的回答，通过点击答案栏下方的"+"按钮来增加多个答案。

    • 机器人回答 - "我们公司的电话号码是080-858-5683。", "有问题就向080-858-5683咨询哟"

  3. 点击"添加"按钮。

  4. 测试
    您现在可以试着在页面右侧的测试对话框中输入您之前添加过的问题中的任何一个。当一个对话模块有多个问题时，这多个问题都通过一个对话模块来进行处理。
    当一个对话模块有多个回答时，聊天机器人用户的输入值和这个对话模块的问题相匹配的话，聊天机器人就会随机给出这多个回答中的一个。

上传对话集

虽然可以通过将对话模块一个一个输入让聊天机器人学习，但是为了方便也可以一次性通过上传对话集(对个对话模块)的方式来让聊天机器人学习。

注意!

上传的对话集文件必须是Excel文件，并且有着要求的格式。

按照以下步骤让聊天机器人学习对话集。

  1. 在页面左侧菜单中选择"管理" -> 选择"对话集"。
  2. 点击上传按钮后填需要填写的信息。
  3. 下载表格样式后，按照样式填入数据(Question列 - 问题, Answer列 - 回答)。
  4. 然后点击"选择文件"按钮来上传文件。
Tip

上传的文件会以新的对话集保存起来，然后要将相应对话集后的"启用"打开才可以开始应用这个对话集。
