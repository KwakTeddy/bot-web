0. 페이스북 회원가입 10초!

Playchat 메인 페이지에서 페이스북 가입(링크)를 클릭하세요.

1. 회사 소개 챗봇 생성 20초!

1. 새 봇 만들기를 클릭하고 -> 빈 봇 클릭하세요.

2. 이름, 설명을 입력한 후 생성을 클릭하세요!

2. 챗봇 학습 20초!

1. 화면의 학습 시작을 클릭하세요!

2.

사용자 입력 : "회사 이름이 뭔가요?" , 챗봇 출력 : "플레이챗 입니다."

사용자 입력 : "회사는 어디에 있나요?" , 챗봇 출력 : "강남역 6번 출구 홍우 빌딩입니다."

사용자 입력 : "회사 전화번호가 어떻게 되나요?" , 챗봇 출력 : "070-858-5683입니다."

3. 추가하기를 클릭하세요.

3. 챗봇 연결 10초!

1. 화면 왼쪽 네비게이션 바의 채널을 클릭하세요.

2. 페이스북 아이콘 연결하기를 클릭하세요.

3. 연결할 페이지 우측 연결하기를 클릭하세요.

4. 해당 페이스북 페이지에서 확인!

4. 끝!

더 자세한 설명을 원하면 여기로!
