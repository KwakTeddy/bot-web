0. 10秒加入Facebook!

在Playchat主页上加入Facebook（链接）

1. 20秒制作公司介绍聊天机器人!

1. 点击新建聊天机器人 -> 选择空白机器人

2. 输入名称, 说明后点击创建按钮!

2. 20秒让聊天机器人学习!

1. 点击页面上的 开始学习按钮!

2.

用户输入 : "你们公司叫什么?" , 机器人回答 : "我们公司名为playchat。"

用户输入 : "你们公司在哪里?" , 机器人回答 : "我们公司在江南地铁站6号出口的宏宇大楼。"

用户输入 : "你们公司的电话号码是多少?" , 机器人回答 : "我们公司的电话号码是070-858-5683。"

3. 点击添加按钮。

3. 10秒连接聊天机器人!

1. 点击页面左侧菜单栏中的频道按钮。

2. 点击Facebook图标下方的连接按钮。

3. 点击在连接页面右侧的连接按钮。

4. 去对应的Facebook主页进行确认!

4. 完成!
